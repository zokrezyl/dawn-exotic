## Why


These builds aim to make Dawn usable on targets that are otherwise inconvenient to bring up, by providing ready-to-use artifacts for tvOS and Linux aarch64 (Raspberry Pi). The goal is to remove build friction and enable running the same WebGPU code across these environments with minimal setup.


As a concrete example, Yetty (https://yetty.dev, https://github.com/zokrezyl/yetty) is a terminal emulator built entirely on WebGPU using Dawn. It runs on both tvOS and Raspberry Pi (aarch64) with a fully GPU-driven rendering pipeline.
